# 概述

Pionex AI Kit 为您提供了一套完整的 AI Agent 基础设施，用于连接 Pionex，包括 **MCP**、**Skills** 和 **CLI**。它支持主流的 AI Agent，如 Cursor、Claude、OpenClaw、Windsurf 和 VSCode。

### 快速开始

| | 组件 | 快速开始 | 指南 | 描述 |
|---|-----------|------------|--------|-------------|
| **MCP** | MCP | [MCP Quickstart](mcp/mcp-quickstart.md) | [MCP Guides](mcp/mcp-guides.md) | 连接 AI agent 与 Pionex 的协议层。 |
| **Skills** | Skills | [Skills Quickstart](skills/skills-quickstart.md) | [Skills Guides](skills/skills-guides.md) | 使 AI agent 能够利用 CLI 高效完成任务的能力。 |
| **CLI** | CLI | [CLI Quickstart](cli/cli-quickstart.md) | [CLI Guides](cli/cli-guides.md) | Pionex API 的命令行界面，通过命令执行操作。 |

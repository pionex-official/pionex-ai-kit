# 概覽

Pionex AI Kit 為您提供一整套連接 Pionex 的 AI Agent 基礎設施，包括 **MCP**、**Skills** 和 **CLI**。支援 Cursor、Claude、OpenClaw、Windsurf 和 VSCode 等主流 AI Agent。

### 快速開始

| | 元件 | 快速入門 | 使用指南 | 說明 |
|---|-----------|------------|--------|-------------|
| **MCP** | MCP | [MCP 快速入門](mcp/mcp-quickstart.md) | [MCP 使用指南](mcp/mcp-guides.md) | 連接 AI Agent 與 Pionex 的協議層。 |
| **Skills** | Skills | [Skills 快速入門](skills/skills-quickstart.md) | [Skills 使用指南](skills/skills-guides.md) | 使 AI Agent 能夠利用 CLI 高效完成任務的能力。 |
| **CLI** | CLI | [CLI 快速入門](cli/cli-quickstart.md) | [CLI 使用指南](cli/cli-guides.md) | 透過命令列介面操作 Pionex API 執行各種操作。 |
